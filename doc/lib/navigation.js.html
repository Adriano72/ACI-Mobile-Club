    open
  






Funzione generica per aprire una finestra

  

  

  

  

    
Params

    

      ctrl
      string
      nome del controller
    

    

      args
      hash
      (opzionale) argomenti da passare al controller in creazione
    

    
Returns

    

      
      Ti.UI.Window
      istanza della Window creata e aperta
    

  



        		
        
function open(ctrl, args) {
    console.log('navigation open ', ctrl);
    var win = Alloy.createController(ctrl, args || {}).getView();
    console.log('win', win);
    exports.Nav.openWindow(win);
    return win;
}

    requireLogin
  






Wrapper di una qualsiasi funzione di navigazione. Implementa la logica per cui una funzione viene eseguita solo se l'utente Ã¨ loggato, altiimenti apre la finestra di login. 
Questo wrapper funziona tramite closure: se ho una funzione myFunc, posso proteggerla generando una nuova funzione myProtectedFunc = requireLogin(myFunc)

  

  

  

  

    
Params

    

      fn
      Function
      funzione da proteggere
    

    
Returns

    

      
      Function
      wrapper della funzione da proteggere, che implementa il meccanismo di login
    

  



        		
        
function requireLogin(fn) {
    return function() {
        if (Alloy.Globals.isLogged) {
            return fn && fn();
        } else {
            return exports.openAccountGateway(fn);
        }
    }
}

    openAccountGateway
  






Apre la schermata AccountGateway in cui l'utente sceglie se loggarsi o registrarsi

  

  

  

  

    
Params

    

      onLogin
      Function
      (opzionale) funzione da eseguire al completamento della login. Solitamente Ã¨ una funzione di reindirizzamento, verrÃ  eventualmente passata alla schermata di login.
    

    
Returns

    

      
      Ti.UI.Window
      istanza della Window creata e aperta
    

  



        		
        
exports.openAccountGateway = function(onLogin) {
    console.log('openAccountGateway');
    return open('account/gateway', {
        onLogin: onLogin
    });
};
